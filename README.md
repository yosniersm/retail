# Retail API

1. Clone repo
2. `npm  install`
3. Copy `.env.template` file and rename to `.env`
4. Change the environment variables
5. Start up database

```
docker-compose up -d
```

6. Run:

```
npm run start:dev
```

7. Setup API seed by making a request to:

```
http://localhost:3000/api/seed
```
